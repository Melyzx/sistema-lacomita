# Sistema Lacomita (RPG VTT)

Este é o VTT (Virtual Tabletop) customizado para as nossas sessões de RPG. O sistema foca em sobrevivência tática, gestão de base e combate.

## 🚀 Funcionalidades
- **Sistema de Sobrevivência:** Mecânicas de fome e sede com penalidades progressivas e morte.
- **Ciclo de Período:** Alternância entre períodos (Vigília e Imolação) que altera o tema do VTT.
- **Gestão de Base (República):** Sistema de upgrades para a base de operações.
- **Sistema de Turnos:** Controle de iniciativa e rodadas automatizado.
- **Dados:** Sistema de rolagem visual interativo.

## ⚙️ Como rodar o projeto

### Pré-requisitos
- Python 3
- Flask (`pip install flask flask-socketio`)

### Execução
1. Abre o terminal na pasta do projeto.
2. Inicia o servidor:
   ```bash
   python3 servidor.py
